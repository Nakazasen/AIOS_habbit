# Sources

Thư mục này quản lý inventory nguồn tri thức.

## Important

Không commit raw source nhạy cảm. `inbox_local_only/` được `.gitignore` bảo vệ.

## Files

- `source_inventory.md`: danh sách source được phép xử lý.
- `excluded_sources.md`: source bị loại và lý do.
- `inbox_local_only/`: nơi tạm để người dùng đặt raw source nếu cần xử lý local.
