# AIOS WorkLens North Star

## What AIOS WorkLens Is
AIOS WorkLens is a local-first work investigation cockpit that turns documents, Excel/CSV files, screenshots, logs, chat snippets, and AI outputs into evidence-backed case files.

## What It Is Not
- Not a NotebookLM clone.
- Not a Cursor or VS Code clone.
- Not a generic second brain.
- Not predictive failure intelligence before evidence discipline exists.
- Not a cloud sync platform.

## Core Pain
Users lose time deciding what happened, what evidence matters, what to do next, and how to hand off a messy work incident.

## Target Users
- Operators, engineers, analysts, support leads, and AI-assisted workers handling live work cases.
- Users who need a daily tool, not another abstract knowledge platform.

## Product Loop
Knowledge → Case → Evidence → Reasoning Map → Action → Outcome → Learning Memory → Better Work

Diễn đạt tiếng Việt trong UI/docs:
Tài liệu nền → Sự việc hằng ngày → Phân tích có bằng chứng → Hành động / giao tiếp → Kết quả thật → Bài học → Trí nhớ công việc → Lần sau làm tốt hơn.

## Principles
- Local-first: private data stays local by default.
- Evidence-first: no claim becomes truth without evidence.
- No fake root cause: hypotheses remain hypotheses until evidence supports them.
- Daily utility beats platform purity.

## Why Not Clone Existing Tools
NotebookLM is useful for source-grounded reading, but WorkLens manages live cases and handovers. Cursor is useful for coding, but WorkLens is for case investigation. A second brain stores knowledge; WorkLens drives action from evidence.
