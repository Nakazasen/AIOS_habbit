# Documentation Archive

Archive giữ evidence lịch sử nhưng không phải hướng dẫn vận hành hiện tại.

- `ux_records/`: UI design gates, implementation audits và owner-smoke records đã đóng.
- `design/`: designs/prompt snapshots superseded bởi architecture/roadmap hiện tại.
- Git history là archive authoritative cho source code đã xóa.

Mỗi file di chuyển giữ nguyên nội dung. Canonical documentation chỉ liên kết vào
archive khi cần truy vết decision/evidence; không dùng archived status làm current state.
